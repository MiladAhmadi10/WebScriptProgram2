CryptoMine Network

In this project we developed a strategic routing protocol that allows a company that goes by the name of CryptoMine to be able to communcate with its other locations, these locations included Toronto, Los Angeles, Chicago, New York. Due to the different protocols that we used such as OSPF and BGP, this allowed for CryptoMine to be connected to each seperate location.
